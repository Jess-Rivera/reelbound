Run Locally 
npm run dev


Run Production from GH and run
npm run build:gh && npm run preview
add /reelbound/ to the link


HOW to push
git add .   // stage changes
git commit -m "insert what i did"
git push origin main


First time clone
git clone https://github.com/Jess-Rivera/reelbound.git


install dependencies
npm install


Pull changes from git
git pull origin main


Common Commands

Purpose	                        Command
See what changed	            git status
Stage all files	                git add .
Commit	                        git commit -m "message"
Push to GitHub	                git push origin main
Pull from GitHub	            git pull origin main
Check commit history	        git log --oneline
